Frequently Asked Questions

  • Is this a real game?

    No. This web site is an assigment for Coderhouse web development course. I decided to use the non-existent "Nightmare on Elm Street" videogame I came up with as the theme for the site.

    The rest of the F.A.Q. will contain questions and answers as if the game was real.

  • When will the game release?

    The estimated release of the Early Access is on Q4 2023. As for the full game, we currently can't give an accurate estimated date.

  • On which platforms will the game be avaible?

    The game will be avaible on PS5, Xbox ONE, and PC. The PC version will be purchasable through Steam and The Epic Game Store. Pre-purchase is also avaible on the Purchase page of this web site.

  • Will the game have microtransactions?

    The game will feature microtransactions. However, these are 100% cosmetic, and won't affect the gameplay in any way.

  • Will the game have any DLC, such as additional characters?

    Currently no DLC are planned for the game. Additional characters that may be added latter in development will be included as a free update for all players.
